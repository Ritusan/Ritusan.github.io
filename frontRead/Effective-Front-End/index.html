/* 借助td实现多列等高 */
.wrapper {
    display: table;
    border-spacing: 20px; /*td间的间距*/
}
.wrapper > div {
    display: table-cell;
    width: 1000px; /*设置很大的宽度，table自动平分宽度*/
    border-radius: 5px;
}

/* 小屏时改成单列显示 */
@media(max-width:500px) {
    .wrapper {
        display: block;
    }
    .wrapper > div {
        display: block;
        width: 100%;
    }

# 1-4尽可能地使用伪元素

伪元素：是一个元素的子元素，并且是inline行内元素，无法用JS获取到伪元素，用伪元素制造视觉上的效果，但是不会增加JS查DOM的负担，它对JS是透明的。

/* :before(::before是HTML5的写法) */
.drop-down:before {
    content: '';
}
.drop-down:after {
    content: '';
}
