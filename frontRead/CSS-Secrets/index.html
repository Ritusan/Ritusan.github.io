# CSS揭秘

# 1.CSS编码技巧

尽量减少重复代码：代码可维护性的最大要素是尽量减少改动时需要编辑的地方。

当某些值相互依赖时，应该把他们的相互关系用代码表达出来。
比如字体和行高，如果改变字号，就得调整行高。但是可以把行高写成：line-height:1.5，这就表示行高是字号的1.5倍。

# 颜色关键字：currentColor

假如想让水平分割线自动与文本的颜色保持一致，可以借用currentColor:

hr {
    height: .5rem;
    background: currentColor;
}

# 灵活的背景定位

background-position扩展语法：允许指定背景图片距离任意角的偏移量，需要在偏移量前面指定关键字。

background: url(img.png)
            no-repeat bottom right #58a;
background-position: right 20px bottom 10px;
1
2
3

background-origin方案：默认情况下background-position是以padding-box为准的，这样边框才不会遮住背景图片，background-origin可以改变这种行为，如果想让背景图片距离边角的偏移量跟内边距保持一致，需要将background-origin设置为content-box。

padding: 10px;
background: url(img.png)
            no-repeat bottom right #58a;
background-origin: content-box;
1
2
3
4

calc()方案：允许指定背景图片距离任意角的偏移量，需要在偏移量前面指定关键字。

注意

calc()函数内部的+和-运算符两侧需要加空格，否则会解析错误。

background: url(img.png)
            no-repeat;
background-position: calc(100% - 20px) calc(100% - 10px);

# 边框内圆角

问题：一个容器，只在内侧有圆角，边框或描边的四个角在外部仍然保持直角的形状，使用一个元素来达成效果。

background tan
border-radius .5rem
padding .625rem
box-shadow 0 0 0 .6rem #655
outline .625rem solid #655

Tip

描边不会跟着元素的圆角走，但是box-shadow却会。
如果把两者结合到一起，box-shadow会刚好填补描边和容器圆角之间的空隙。

# 条纹背景

Tip

如果某个色标的位置值比整个列表中在它之前的色标的位置值都要小，则该色标的位置值会被设置为它前面所有色标位置值的最大值。

水平条纹：如果把第二个色标的位置值设置为0，那它的位置就总是会被浏览器调整为前一个色标的位置值。

height 120px
background linear-gradient(#fb3 30%, #58a 0, #58a 66.6%, yellowgreen 0)
background-size 100% 30px
1
2
3

垂直条纹：垂直条纹的代码和水平条纹几乎一样，差别在于：需要在开头加上一个额外的参数来指定渐变的方向。还需要将background-size的值颠倒一下。

height 120px
background linear-gradient(to right, #fb3 50%, #58a 0, #58a 66.6%, yellowgreen 0)
background-size 30px 100%
1
2
3

斜向条纹：利用重复线性渐变。

height 120px
background repeating-linear-gradient(45deg, #fb3 , #fb3 15px, #58a 0, #58a 30px)
1
2

同色系条纹：把最深的颜色指定为背景色，同时把半透明白色的条纹叠加在背景色之上来得到浅色条纹。

height 120px
background #58a
background-image repeating-linear-gradient(30deg,hsla(0,0%,100%,.1) 0 15px, transparent 0 30px)
1
2
3

# 椭圆

border-radius

可以单独指定水平和垂直半径，需要用一个斜杠(/)分隔两个值。这时会产生一个椭圆，椭圆的水平和垂直半径就是在border-radius中指定的值。

# 切角效果

CSS3渐变

利用渐变的一大特性：渐变可以接受一个角度作为方向，而且色标的信息也可以是绝对的长度值。
渐变需要把一个透明色标放在切角处，然后在相同位置设置另一个色标，并且把它的颜色设置为我们想要的背景色。

height 120px
background #b4b4ff
background linear-gradient(-45deg,transparent 15px, #b4b4ff 0)

radial-gradient(circle at bottom right,transparent 15px,#b4b4ff 0) bottom right,
