# VSCode常用快捷键

常用操作 说明
cmd + P 快速打开文件
cmd + W 关闭tab页
cmd + 1 切换到第1个tab
control + tab 切换同一编辑器不同tab页
cmd + / 添加行注释
shift + alt + A 添加块注释
操作行 说明
alt + up/down 代码上下移动一行
shift + alt + up/down 代码向上向下复制一行
cmd + enter 在当前行下方插入一行
cmd + shift + enter 在当前行上方插入一行
cmd + shift + K 删除当前行
control + G 跳转到某一行
跳转 说明
fn + ⬅️ 跳到当前行的头部
fn + ➡️ 跳到当前行的尾部
cmd + ⬅️ 滚动到页头
cmd + ➡️ 滚动到页尾
alt + 点击 插入多光标
cmd + U 撤销上一个光标操作
cmd +/- 放大缩小字体
其它 说明
control + cmd + 空格 插入表情
cmd + \ 打开与关闭侧边栏
cmd + K + P 复制当前文件的路径
control + cmd + F 编辑器切换全屏
shift + cmd + V 切换Markdown预览
cmd + K + V Markdown预览到一边
control + ` 显示集成终端
