# markdown

相对于复杂的 html 标记语言来说，markdown 是一种轻量级的标记语言。

# 1. 标题：#

# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

# 2. 无序列表：-/+/*

- red
+ green
* blue

  • red

  • green

  • blue

# 3. 有序列表：数字+.

  1. red

  2. green

  3. blue

# 4. 引用：>

markdown

# 5. 强调

两个*/：加粗
一个*/：斜体
两个~：删除

**red**
*red*
~~red~~

red

red

red

# 8. 分割线

在一行中用三个以上的 * 、- 、_ 来建立一个分隔线，行内不能有其他东西。也可以在符号间插入空格。效果是一样的。

VSCode编辑markdown文件

在vscode中编写Markdown文件时，会遇到明明按回车换行了但是预览的时候却没有换行的情况，这时在需要换行的地方多按两次空格键，就会换行
