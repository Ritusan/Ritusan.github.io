# 海子的诗

愿你有灿烂前程，也愿你能读一下海子的诗

给每一条河每一座山取一个温暖的名字
陌生人，我也为你祝福
愿你有一个灿烂的前程
愿你有情人终成眷属
愿你在尘世获得幸福
我只愿面朝大海，春暖花开

远方就是你一无所有的地方

沉默是山洞
沉默是山洞里一大桶黄金
沉默是因为爱情
